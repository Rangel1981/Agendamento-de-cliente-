# 📅 Sistema de Agendamento de Serviços

Este é um projeto desenvolvido em **Python** focado em aplicar conceitos de Programação Orientada a Objetos (POO) e persistência de dados. O sistema permite gerenciar cadastros de clientes e organizar horários de serviços, garantindo que não haja conflitos de agenda.

## 🚀 Funcionalidades

- **Cadastro de Clientes:** Registro de nome e telefone.
- **Gestão de Agendamentos:** Vinculação de um serviço a um cliente com data e hora.
- **Validação de Conflitos:** O sistema impede dois agendamentos no mesmo horário e data.
- **Listagem Dinâmica:** Visualização de todos os serviços marcados.
- **Persistência em JSON:** Os dados são salvos localmente (em desenvolvimento).

## 🛠️ Tecnologias Utilizadas

- **Python 3.x**
- **Biblioteca JSON** (nativa)
- **Paradigma de POO** (Classes, Composição e Métodos)
